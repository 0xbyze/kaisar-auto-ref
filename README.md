# kaisar-auto-ref
Automatic referral script for Kaisar using captcha solver and proxies.


## Installation

Make sure you have nodejs installed.

```bash
npm install
```

## Proxy format
Below is the format proxy

```bash
http://user:pass@host:port
```

## Usage

1. run auto ref
```bash
npm start
```

2. Split token.
```bash
npm run split
```
